# Estrutura da Ficha 03

A **Ficha 03** contém 8 exercícios práticos organizados da seguinte forma:

### 📂 Bloco 1 - Manipulação de Strings e Tuplos
* **exercicio_01_first_last.py**: Decomposição de strings e extração de caracteres em tuplos.
* **exercicio_02_palindromo.py**: Lógica de verificação de simetria em cadeias de caracteres.
* **exercicio_05_remove_all.py**: Filtragem e remoção seletiva de caracteres em strings.

### 📂 Bloco 2 - Processamento de Sequências e Índices
* **exercicio_03_odd_even.py**: Reordenação de elementos com base em posições ímpares e pares.
* **exercicio_04_duplicate.py**: Transformação de sequências através da duplicação de elementos.
* **exercicio_07_search_indices.py**: Localização de primeira e última ocorrência de valores.

### 📂 Bloco 3 - Análise de Dados e Interfaces de Menu
* **exercicio_06_extremos.py**: Processamento de coleções para identificar limites de comprimento.
* **exercicio_08_menu_pessoas.py**: Sistema completo de gestão de dados (Lista de Tuplos) com interface de menu funcional.
